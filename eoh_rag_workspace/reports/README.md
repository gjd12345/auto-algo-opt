# Reports Layout

This directory keeps only the current, human-readable research evidence.
Large raw runs and older report generations are archived outside this tree.

## Current directories

- `auto_experiment_reports/`: current TOCC experiment summaries, decks, code records, and compact evidence tables.
- `paper_notes/`: paper reading notes, related-work drafts, and method notes used by the current TOCC research direction.

## Archived material

Historical Guarded EOH-Go tables, figures, paper drafts, and older TOCC trial reports were moved to:

- `archived_experiments/reports_20260619/`

Raw per-run directories named `run_*` are not tracked in git. They are stored locally under:

- `archived_experiments/raw_runs_20260619/`

This split keeps the codebase readable while preserving reproducibility evidence for later inspection.
